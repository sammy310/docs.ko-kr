# <a name="-help---visual-basic"></a>-help, -? (Visual Basic)
컴파일러 옵션을 표시합니다.  
  
## <a name="syntax"></a>구문  
  
```console  
-help  
```

또는  

```console
-?  
```  
  
## <a name="remarks"></a>설명  
 이 옵션을 컴파일에 포함하면 출력 파일이 생성되지 않고 컴파일이 수행되지 않습니다.  
  
> [!NOTE]
> Visual Studio 개발 환경 내에서는 `-help` 옵션을 사용할 수 없습니다. 명령줄에서 컴파일하는 경우에만 사용할 수 있습니다.  
  
## <a name="example"></a>예제  
 다음 코드는 명령줄의 도움말을 표시합니다.  
  
```console  
vbc -help  
```
