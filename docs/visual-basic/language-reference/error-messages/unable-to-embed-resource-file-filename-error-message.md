# <a name="bc30143-unable-to-embed-resource-file-filename-error-message"></a>BC30143: ' ' 리소스 파일을 포함할 수 없습니다 \<filename> . \<error message>

Visual Basic 컴파일러는 어셈블리 링커 (Al.exe (Alink) 라고도 함)를 호출 하 여 매니페스트로 어셈블리를 생성 합니다. 링커가 네이티브 COM+ 리소스 파일을 어셈블리에 직접 포함하는 동안 오류를 보고했습니다.

 **오류 ID:** BC30143

## <a name="to-correct-this-error"></a>이 오류를 해결하려면

1. 따옴표 붙은 오류 메시지를 검사 하 고 [Al.exe](../../../framework/tools/al-exe-assembly-linker.md)항목을 참조 하세요. 추가 설명과 조언을 제공 합니다.

2. 오류가 계속 발생하면 해당 상황에 대한 정보를 수집하여 Microsoft 기술 지원 서비스에 알립니다.
