# <a name="loop-statements-are-not-valid-in-the-immediate-window"></a>직접 실행 창에서는 Loop 문을 사용할 수 없습니다.

`Loop`**직접 실행** 창에서는 문을 사용할 수 없습니다.  
  
 **오류 ID:** BC30708  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 디버깅 코드에서 `Loop` 문을 제거합니다.
