# <a name="namespace-or-type-name-has-already-been-imported"></a>Namespace 형식 또는 '\<이름 >'에 이미 가져왔습니다
네임스페이스를 두 번 이상 가져오려고 했습니다.  
  
 **오류 ID:** BC31051  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   중복 `Imports` 문을 제거합니다.
