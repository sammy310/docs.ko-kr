# <a name="character-sequence----is-not-allowed-in-an-xml-comment"></a>XML 주석에는 문자 시퀀스 '--'을 사용할 수 없습니다.

문자 시퀀스 '--'를 포함하는 XML 주석 리터럴은 지원되지 않습니다.  
  
 **오류 ID:** BC31171  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 주석 리터럴에서 문자 시퀀스 '--'를 제거합니다.
