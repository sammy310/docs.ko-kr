# <a name="error-structurename1-contains-structurename2"></a>\<오류 >: '\<structurename1 >' 포함 '\<structurename2 >'
구조체가 자신을 해당 멤버 중 하나로 지정하는 경우 순환 구조 정의가 발생합니다.  
  
 **오류 ID:** BC30293  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 구조체 멤버의 이름을 변경합니다.
