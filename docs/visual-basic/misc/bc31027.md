# <a name="unable-to-open-file-filename-error"></a>파일을 열 수 없습니다. '\<파일 이름 >': \<오류 >
이 프로젝트에서 파일을 여는 동안 오류가 발생했습니다.  
  
 **오류 ID:** BC31027  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 파일이 있고, 손상되었거나 다른 애플리케이션에 의해 잠기지 않았는지 확인합니다.
