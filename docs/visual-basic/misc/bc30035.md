# <a name="syntax-error-visual-basic-error"></a>구문 오류(Visual Basic 오류)

소스 코드 줄에서 일반 구문 오류를 생성합니다.  
  
 **오류 ID:** BC30035  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 소스 코드 줄에서 사용하는 각 키워드에 대한 설명서를 검사합니다. 설명서의 예제 코드를 이 오류를 생성한 소스 줄과 비교합니다.  
  
2. 오류의 원인을 식별할 수 없으면 해당 상황에 대한 정보를 수집하여 Microsoft 기술 지원 서비스에 알립니다.
