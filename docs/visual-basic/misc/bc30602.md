# <a name="interface-members-must-be-methods-properties-events-or-type-definitions"></a>인터페이스 멤버는 메서드, 속성, 이벤트 또는 형식 정의여야 합니다.

중첩 형식, 메서드, 속성 및 이벤트만 인터페이스 멤버일 수 있습니다.  
  
 **오류 ID:** BC30602  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 항목이 메서드, 속성, 이벤트 또는 열거형 등의 중첩 형식인지 확인합니다.
