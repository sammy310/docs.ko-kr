# <a name="parameter-parametername-of-methodname-already-has-a-matching-argument"></a>매개 변수 '\<parametername >'의 '\<methodname >'는 일치 하는 인수가 이미 있습니다.
인수가 동일한 프로시저 호출에서 두 번 이상 이름으로 전달됩니다.  
  
 **오류 ID:** BC30274  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   인수 이름 중 하나의 철자가 잘못되었는지 확인합니다.  
  
-   인수 목록에서 중복된 인수를 제거합니다.
