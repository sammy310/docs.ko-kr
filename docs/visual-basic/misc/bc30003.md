# <a name="next-expected"></a>'Next'가 필요합니다.
`On Error Resume` 구문이 종료 `Next`없이 발생합니다.  
  
 **오류 ID:** BC30003  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
-   `Next` 문의 끝에 `On Error` 키워드를 추가합니다.
