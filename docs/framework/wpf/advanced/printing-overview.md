 <span data-ttu-id="0cde3-206">스풀 크기 감소는 다음과 같은 여러 메커니즘을 통해 수행됩니다.</span><span class="sxs-lookup"><span data-stu-id="0cde3-206">Spool size reduction is performed through several mechanisms:</span></span>  
  
- <span data-ttu-id="0cde3-207">**글꼴 하위 집합**.</span><span class="sxs-lookup"><span data-stu-id="0cde3-207">**Font subsetting**.</span></span> <span data-ttu-id="0cde3-208">실제 문서내에 사용된 문자만 XPS 파일에 저장됩니다.</span><span class="sxs-lookup"><span data-stu-id="0cde3-208">Only characters used within the actual document are stored in the XPS file.</span></span>  
  
- <span data-ttu-id="0cde3-209">**고급 그래픽 지원**.</span><span class="sxs-lookup"><span data-stu-id="0cde3-209">**Advanced Graphics Support**.</span></span> <span data-ttu-id="0cde3-210">투명도 및 그라데이션 프리미티브에 대한 기본 지원은 XPS 문서에서 콘텐츠의 래스터화를 방지합니다.</span><span class="sxs-lookup"><span data-stu-id="0cde3-210">Native support for transparency and gradient primitives avoids rasterization of content in the XPS Document.</span></span>

- <span data-ttu-id="0cde3-213">**ZIP 압축**.</span><span class="sxs-lookup"><span data-stu-id="0cde3-213">**ZIP compression**.</span></span> <span data-ttu-id="0cde3-214">모든 XPS 문서는 ZIP 압축을 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="0cde3-214">All XPS documents use ZIP compression.</span></span>
