# <a name="icordebugchainismanaged-method"></a>ICorDebugChain::IsManaged 메서드
이 체인이 관리 코드를 실행 하 고 있는지 여부를 나타내는 값을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT IsManaged (  
    [out] BOOL               *pManaged  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `pManaged`  
 [out]이 체인이 관리 코드를 실행 하는 경우에 `true` 합니다. 그렇지 않으면 `false`합니다.
