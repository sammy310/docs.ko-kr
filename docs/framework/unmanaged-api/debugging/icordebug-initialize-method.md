# <a name="icordebuginitialize-method"></a>ICorDebug::Initialize 메서드
`ICorDebug` 개체를 초기화합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Initialize ();  
```  
  
## <a name="remarks"></a>설명  
 디버거는 만들 때 `Initialize` 를 호출 하 여 디버깅 서비스를 초기화 해야 합니다. 이 메서드는의 `ICorDebug` 다른 메서드를 호출 하기 전에 호출 해야 합니다.
