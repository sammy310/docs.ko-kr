### <a name="endpoint-addresses-for-iis-hosted-services"></a><span data-ttu-id="95440-169">IIS에서 호스팅되는 서비스의 엔드포인트 주소</span><span class="sxs-lookup"><span data-stu-id="95440-169">Endpoint Addresses for IIS-Hosted Services</span></span>
