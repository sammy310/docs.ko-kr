## <a name="how-snapshot-isolation-and-row-versioning-work"></a><span data-ttu-id="83dd9-160">스냅샷 격리 및 행 버전 관리의 작동 방법</span><span class="sxs-lookup"><span data-stu-id="83dd9-160">How Snapshot Isolation and Row Versioning Work</span></span>
