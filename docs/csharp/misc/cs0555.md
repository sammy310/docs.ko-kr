<span data-ttu-id="70f13-103">사용자 정의 연산자는 바깥쪽 형식의 개체를 가져와서 바깥쪽 형식의 개체로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="70f13-103">User-defined operator cannot take an object of the enclosing type and convert to an object of the enclosing type</span></span>
