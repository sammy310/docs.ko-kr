# <a name="compiler-error-cs1660"></a><span data-ttu-id="9f817-102">컴파일러 오류 CS1660</span><span class="sxs-lookup"><span data-stu-id="9f817-102">Compiler Error CS1660</span></span>
<span data-ttu-id="9f817-103">무명 메서드 블록이 대리자 형식이 아니므로 'type' 형식으로 변환할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="9f817-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="9f817-104">이 오류는 무명 메서드 블록을 대리자 형식이 아닌 형식에 할당하거나 변환하려고 하는 경우 발생합니다.</span><span class="sxs-lookup"><span data-stu-id="9f817-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="9f817-105">다음 샘플에서는 CS1660을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="9f817-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
