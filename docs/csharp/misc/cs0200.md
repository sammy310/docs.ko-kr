# <a name="compiler-error-cs0200"></a>컴파일러 오류 CS0200
'property' 속성 또는 인덱서는 읽기 전용이므로 할당할 수 없습니다.  
  
 [property](../programming-guide/classes-and-structs/using-properties.md)에 값을 할당하려고 했지만 property에 set 접근자가 없습니다. set 접근자를 추가하여 오류를 해결합니다. 자세한 내용은 [읽기/쓰기 속성을 선언 하 고 사용 하는 방법](../programming-guide/classes-and-structs/how-to-declare-and-use-read-write-properties.md)을 참조 하세요.
  
## <a name="example"></a>예제  
 다음 샘플에서는 CS0200을 생성합니다.  
  
```csharp  
// CS0200.cs  
public class MainClass  
{  
    // private int _mi;  
    int I  
    {  
        get  
        {  
            return 1;  
        }  
  
        // uncomment the set accessor and declaration for _mi  
        /*  
        set  
        {  
            _mi = value;  
        }  
        */  
    }  
  
    public static void Main ()  
    {  
        MainClass II = new MainClass();  
        II.I = 9;   // CS0200  
    }  
}  
```
