# <a name="compiler-error-cs0036"></a><span data-ttu-id="ad14d-102">컴파일러 오류 CS0036</span><span class="sxs-lookup"><span data-stu-id="ad14d-102">Compiler Error CS0036</span></span>
<span data-ttu-id="ad14d-103">out 매개 변수에는 '[In]' 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="ad14d-103">An out parameter cannot have the '[In]' attribute</span></span>  
  
 <span data-ttu-id="ad14d-104">현재 **In** 특성은 [out](../language-reference/keywords/out-parameter-modifier.md) 매개 변수에서 허용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="ad14d-104">Currently, the **In** attribute is not permitted on an [out](../language-reference/keywords/out-parameter-modifier.md) parameter.</span></span>  
  
 <span data-ttu-id="ad14d-105">다음 샘플에서는 CS0036을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="ad14d-105">The following sample generates CS0036:</span></span>  
  
```csharp  
// CS0036.cs  
  
using System;  
using System.Runtime.InteropServices;  
  
public class MyClass  
{  
   public static void TestOut([In] out char TestChar)   // CS0036  
   // try the following line instead  
   // public static void TestOut(out char TestChar)  
   {  
      TestChar = 'b';  
      Console.WriteLine(TestChar);  
   }  
  
   public static void Main()  
   {  
      char i;           //variable to receive the value  
      TestOut(out i);   // the arg must be passed as out  
      Console.WriteLine(i);  
   }  
}  
```
