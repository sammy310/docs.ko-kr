# <a name="compiler-error-cs0065"></a><span data-ttu-id="7affe-102">컴파일러 오류 CS0065</span><span class="sxs-lookup"><span data-stu-id="7affe-102">Compiler Error CS0065</span></span>

<span data-ttu-id="7affe-103">'event': 이벤트 속성에는 add 및 remove 접근자가 둘 다 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="7affe-103">'event' : event property must have both add and remove accessors</span></span>

<span data-ttu-id="7affe-104">필드가 아닌 이벤트에는 두 액세스 메서드가 모두 있어야 합니다.</span><span class="sxs-lookup"><span data-stu-id="7affe-104">An event that is not a field must have both access methods.</span></span>

<span data-ttu-id="7affe-105">다음 샘플에서는 CS0065를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7affe-105">The following sample generates CS0065:</span></span>

```csharp
// CS0065.cs
using System;
public delegate void EventHandler(object sender, int e);
public class MyClass
{
   public event EventHandler Click   // CS0065,
   {
      // to fix, uncomment the add and remove definitions
      /*
      add
      {
         Click += value;
      }
      remove
      {
         Click -= value;
      }
      */
   }

   public static void Main()
   {
   }
}
```
