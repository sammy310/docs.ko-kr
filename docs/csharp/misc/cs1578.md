 <span data-ttu-id="05e7e-104">[#line](../language-reference/preprocessor-directives/preprocessor-line.md) 지시문 뒤에는 큰따옴표로 묶은 파일 이름 또는 한 줄로 된 주석만 허용됩니다.</span><span class="sxs-lookup"><span data-stu-id="05e7e-104">After a [#line](../language-reference/preprocessor-directives/preprocessor-line.md) directive, only a file name (in double quotation marks) or a single-line comment is allowed.</span></span>

```csharp

   static void Main()  
   {  
      #line 101 abc.cs   // CS1578

      intt i;          // error will be reported on line 101  
   }  
}  
```
