 <span data-ttu-id="69af8-104">[event](../language-reference/keywords/event.md) 가 필드로 선언되지 않은 경우 **add** 및 **remove** 접근자 함수를 모두 정의해야 합니다.</span><span class="sxs-lookup"><span data-stu-id="69af8-104">If your [event](../language-reference/keywords/event.md) is not declared as a field, it must define both **add** and **remove** accessor functions.</span></span>  
  
 <span data-ttu-id="69af8-105">다음 샘플에서는 CS1055를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="69af8-105">The following sample generates CS1055:</span></span>  
  
```csharp  
// CS1055.cs  
delegate void del();

{  
   public event del MyEvent  
   {  
      int i;   // CS1055  
      // uncomment accessors and delete previous line to resolve  
      // add  
      // {  
      //    MyEvent += value;  
      // }  
      // remove  
      // {  
      //    MyEvent -= value;  
      // }  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
