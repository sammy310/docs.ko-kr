 <span data-ttu-id="7365d-105">예를 들어 명시적 인터페이스 정의에 일부 특성을 사용할 수 없습니다.</span><span class="sxs-lookup"><span data-stu-id="7365d-105">For example, you cannot use some attributes on an explicit interface definition.</span></span> <span data-ttu-id="7365d-106">다음 샘플에서는 CS0577을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7365d-106">The following sample generates CS0577:</span></span>  
  
```csharp  
// CS0577.cs  
// compile with: /target:library

{  
   void m();  
}  
  
public class MyClass : I  
{  
   [System.Diagnostics.Conditional("a")]   // CS0577  
   void I.m() {}  
}  
```
