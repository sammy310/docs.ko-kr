# <a name="compiler-error-cs0568"></a>컴파일러 오류 CS0568

구조체는 매개 변수가 없는 명시적 생성자를 포함할 수 없습니다.  
  
 각 [구조체](../language-reference/builtin-types/struct.md) 에는 개체를 0으로 초기화 하는 매개 변수가 없는 생성자가 이미 있습니다. 따라서 구조체에 대해 만들 수 있는 생성자는 하나 이상의 매개 변수를 사용해야 합니다.  
  
 다음 샘플에서는 CS0568을 생성합니다.  
  
```csharp  
// CS0568.cs  
public struct ClassY  
{  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor  
   // Try following instead:  
   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}  
  
public class ClassX  
{  
   public static void Main()  
   {  
   }  
}  
```
