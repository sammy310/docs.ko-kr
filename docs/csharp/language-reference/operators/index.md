## <a name="operator-associativity"></a><span data-ttu-id="95468-148">연산자 결합성</span><span class="sxs-lookup"><span data-stu-id="95468-148">Operator associativity</span></span>

- <span data-ttu-id="95468-150">‘왼쪽 결합성이 있는’ 연산자는 왼쪽에서 오른쪽으로 계산됩니다. </span><span class="sxs-lookup"><span data-stu-id="95468-150">*Left-associative* operators are evaluated in order from left to right.</span></span> <span data-ttu-id="95468-151">[대입 연산자](assignment-operator.md) 및 [null 병합 연산자 ](null-coalescing-operator.md)를 제외하고, 모든 이진 연산자는 왼쪽 결합성이 있습니다.</span><span class="sxs-lookup"><span data-stu-id="95468-151">Except for the [assignment operators](assignment-operator.md) and the [null-coalescing operators](null-coalescing-operator.md), all binary operators are left-associative.</span></span> <span data-ttu-id="95468-152">예를 들어, `a + b - c`는 `(a + b) - c`로 계산됩니다.</span><span class="sxs-lookup"><span data-stu-id="95468-152">For example, `a + b - c` is evaluated as `(a + b) - c`.</span></span>
- <span data-ttu-id="95468-153">‘오른쪽 결합성이 있는’ 연산자는 오른쪽에서 왼쪽으로 계산됩니다. </span><span class="sxs-lookup"><span data-stu-id="95468-153">*Right-associative* operators are evaluated in order from right to left.</span></span> <span data-ttu-id="95468-154">대입 연산자, null 병합 연산자 및 [조건 연산자 `?:`](conditional-operator.md)는 오른쪽 결합성이 있습니다.</span><span class="sxs-lookup"><span data-stu-id="95468-154">The assignment operators, the null-coalescing operators, and the [conditional operator `?:`](conditional-operator.md) are right-associative.</span></span> <span data-ttu-id="95468-155">예를 들어, `x = y = z`는 `x = (y = z)`로 계산됩니다.</span><span class="sxs-lookup"><span data-stu-id="95468-155">For example, `x = y = z` is evaluated as `x = (y = z)`.</span></span>

<span data-ttu-id="95468-156">괄호를 사용하여 연산자 결합성에 따라 주어진 계산 순서를 변경합니다.</span><span class="sxs-lookup"><span data-stu-id="95468-156">Use parentheses to change the order of evaluation imposed by operator associativity:</span></span>
