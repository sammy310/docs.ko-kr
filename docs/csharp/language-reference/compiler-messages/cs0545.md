## <a name="example"></a><span data-ttu-id="7ebdc-111">예제</span><span class="sxs-lookup"><span data-stu-id="7ebdc-111">Example</span></span>  
 <span data-ttu-id="7ebdc-112">다음 샘플에서는 CS0545를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="7ebdc-112">The following sample generates CS0545.</span></span>  
  
```csharp  
// CS0545.cs  
// compile with: /target:library  
// CS0545  
public class a  
{  
   public virtual int i  
   {  
      set {}  
  
      // Uncomment the following line to resolve.  
      // get { return 0; }  
   }  
}  
  
public class b : a  
{  
   public override int i  
   {  
      get { return 0; }  
      set {}   // OK  
   }  
}  
```
