## <a name="example"></a><span data-ttu-id="fe648-115">예제</span><span class="sxs-lookup"><span data-stu-id="fe648-115">Example</span></span>  

 <span data-ttu-id="fe648-116">다음 예제에서는 CS0433을 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="fe648-116">The following example generates CS0433.</span></span>  
  
```csharp

public class Test
{  
   public static void Main()
   {  
      AggPubImpAggPubImp n6 = new AggPubImpAggPubImp();   // CS0433  
   }  
}  
```
