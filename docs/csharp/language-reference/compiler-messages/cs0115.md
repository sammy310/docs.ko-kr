## <a name="example"></a><span data-ttu-id="a1212-106">예제</span><span class="sxs-lookup"><span data-stu-id="a1212-106">Example</span></span>

 <span data-ttu-id="a1212-107">다음 샘플에서는 CS0115를 생성합니다.</span><span class="sxs-lookup"><span data-stu-id="a1212-107">The following sample generates CS0115.</span></span> <span data-ttu-id="a1212-108">다음 두 가지 방법 중 하나로 CS0115를 해결할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="a1212-108">You can resolve CS0115 in one of two ways:</span></span>

- <span data-ttu-id="a1212-109">`override`의 메서드에서 `MyClass2` 키워드를 제거합니다.</span><span class="sxs-lookup"><span data-stu-id="a1212-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>

- <span data-ttu-id="a1212-110">`MyClass1`을 `MyClass2`의 기본 클래스로 사용합니다.</span><span class="sxs-lookup"><span data-stu-id="a1212-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>

```csharp
// CS0115.cs
namespace MyNamespace
{
    abstract public class MyClass1
    {
        public abstract int f();
    }

    abstract public class MyClass2
    {
        public override int f()   // CS0115
        {
            return 0;
        }

        public static void Main()
        {
        }
    }
}
```
