## <a name="low-latency-settings"></a><span data-ttu-id="1ab15-107">짧은 대기 시간 설정</span><span class="sxs-lookup"><span data-stu-id="1ab15-107">Low latency settings</span></span>

<span data-ttu-id="1ab15-108">"짧은" 대기 시간 설정을 사용하면 가비지 수집기의 애플리케이션 개입이 줄어듭니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-108">Using a "low" latency setting means the garbage collector intrudes less in your application.</span></span> <span data-ttu-id="1ab15-109">가비지 수집은 좀 더 적은 양의 메모리를 회수합니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-109">Garbage collection is more conservative about reclaiming memory.</span></span>

<span data-ttu-id="1ab15-110"><xref:System.Runtime.GCLatencyMode?displayProperty=nameWithType> 열거형은 두 개의 짧은 대기 시간 설정을 제공합니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-110">The <xref:System.Runtime.GCLatencyMode?displayProperty=nameWithType> enumeration provides two low latency settings:</span></span>

- <span data-ttu-id="1ab15-111">[GCLatencyMode.LowLatency](xref:System.Runtime.GCLatencyMode.LowLatency)는 2세대 컬렉션을 사용하지 않고 0세대 및 1세대 컬렉션만 수행합니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-111">[GCLatencyMode.LowLatency](xref:System.Runtime.GCLatencyMode.LowLatency) suppresses generation 2 collections and performs only generation 0 and 1 collections.</span></span> <span data-ttu-id="1ab15-112">짧은 시간 동안에만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-112">It can be used only for short periods of time.</span></span> <span data-ttu-id="1ab15-113">긴 시간 동안 사용하여 시스템의 메모리 사용량이 많을 경우 가비지 수집기가 애플리케이션을 잠시 중지하고 시간 결정적 작업을 방해할 수 있는 수집을 트리거합니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-113">Over longer periods, if the system is under memory pressure, the garbage collector will trigger a collection, which can briefly pause the application and disrupt a time-critical operation.</span></span> <span data-ttu-id="1ab15-114">이 설정은 워크스테이션 가비지 컬렉션에서만 사용할 수 있습니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-114">This setting is available only for workstation garbage collection.</span></span>

<span data-ttu-id="1ab15-118">대기 시간이 짧은 기간 중에는 다음이 발생하지 않는 한 2세대 컬렉션이 사용되지 않습니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-118">During low latency periods, generation 2 collections are suppressed unless the following occurs:</span></span>

- <span data-ttu-id="1ab15-120">애플리케이션 코드는 <xref:System.GC.Collect%2A?displayProperty=nameWithType> 메서드를 호출하고 `generation` 매개 변수에 대해 2를 지정하여 수집을 줄입니다.</span><span class="sxs-lookup"><span data-stu-id="1ab15-120">Application code induces a collection by calling the <xref:System.GC.Collect%2A?displayProperty=nameWithType> method and specifying 2 for the `generation` parameter.</span></span>
